***SOBRE O PROJETO:***

  - É necessário ter o NodeJS na máquina.

***INSTALAÇÃO***

-No terminal, execute:
    
      npm install -g expo-cli
      
- Para baixar o repositório, utilize o comando:

      git clone https://github.com/anajuca/CalculadoraIMC.git
- Entre no diretório do repositório
  
      cd CalculadoraIMC

  
- Inicie o projeto

      npx expo start --tunnel
- Caso ele peça para instalar ngrok, digite "y" e aperte enter.

- Escaneie o QR-Code no seu celular usando o aplicativo "Expo Go".

-Pronto!

***FUNCIONAMENTO E OBJETIVOS***
- O projeto utiliza da linguagem JS para fazer um programa que calcula o IMC de um usuário de acordo com os dados fornecidos, além de informar a ele sua classificação (abaixo do peso, peso ideal, obeso etc) e o peso ideal que deve buscar.
